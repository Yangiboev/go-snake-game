## Running Locally

1. Fetch the repo
```bash
git clone https://github.com/Yangiboev/go-snake-game.git
```

2. Download the dependencies
```bash
cd go-snake-game
make tidy
```

3. Starting the game
```bash
go run cmd/main.go
```
